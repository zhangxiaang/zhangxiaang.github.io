About Mysql

有时候开启server的时候爆出

异常，正确的解法是把/usr/local/var/mysql/下的文件以.err结尾的文件都给删除掉
重新开启mysql.server start可以直接开启服务

另外呢 经常忘记密码
修改密码流程如下:

0. 关掉所有的mysqld的服务(在activity monitor.app里面检索mysqld点击close即可)
1. mysqld --skip-grant-tables --user=USERNAME & (USERNAME=你的root用户名)
2. 这个时候就可以直接输入mysql进入到数据库中了。
3. update mysql.user SET authentication_string = PASSWORD('NewPassord'), password_expired = 'N'
WHERE User = 'root' AND Host = 'localhost';
4. 如果返回的是一个Rows matched: 1  Changed: 1  即可继续执行FLUSH PRIVILEGES; 更新内容了。
5. 接着只需要关闭mysqld，使用mysql.server start一个新的mysqld出来即可(可能会遇到without updating PID file按照上叙即可解决)

Some about ThinkPhp链接数据库的问题:

0. 如果你的table里面有大小写混写的name的话，在config.php里面添加一个配置'DB_PARAMS'                =>  array(\PDO::ATTR_CASE => \PDO::CASE_NATURAL),即可解决问题
1. 如果在链接数据库的时候一直遇到如下问题: SQLSTATE[HY000] [2002] No such file or directory  查询知道是在DbDriver这里链接数据库出了问题，肯定是参数错了，但是奇葩的是解决方案是把 'DB_HOST'->'localhost' 改成 'DB_HOST'->'127.0.0.1'就解决这个问题 ， 也许是这个版本中的DNS并没有根据localhost来解析对应的ip地址吧
